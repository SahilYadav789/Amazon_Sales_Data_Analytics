# Amazon Sales Analytics Dashboard
 Project Overview

 Developed an end-to-end e-commerce analytics project using Python and Power BI to analyze       Amazon sales performance, customer demand, and operational efficiency.

# Tools & Technologies
1. Python
2. Pandas
3. NumPy
4. Matplotlib
5. Power BI

# Key Business Insights
Top categories (Set & Kurta) contributed majority of revenue
Revenue peaked in April and declined in subsequent months
High cancellation volume impacted business performance
Product demand heavily concentrated in a few categories.

# Dashboard Features
Revenue KPI tracking
Monthly sales trend analysis
Category-wise revenue analysis
Order fulfillment status breakdown
Product demand analysis

# Data Cleaning & Processing
Removed null values from critical columns
Handled inconsistent records
Filtered zero-value transactions
Created calculated KPIs for analysis
